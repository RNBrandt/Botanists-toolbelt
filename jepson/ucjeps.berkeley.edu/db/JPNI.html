Jepson Place Name Index

The Jepson Place Name Index contains approximately 41,000 place names found on botanical collection labels. Over a period of approximately 50 years, staff and associates of the Jepson Herbarium have researched these place names using maps in libraries and archives at Berkeley and elsewhere. The JPNI has been maintained by the Jepson Herbarium as a card file for reference purposes.
